# vue3-wysiwyg

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Launch dist
```
serve -s dist
```

### Deploy to Firebase hosting (build + deploy)

```
npm run deploy
```
